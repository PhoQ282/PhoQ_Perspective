# PhoQ Perspective
A lightweight **hold-to-zoom** third/first-person camera for FiveM.  
This script provides a smooth zoom effect while holding the **middle mouse button**, without any freecam or noclip systems attached.

## Features
- **Hold to Zoom** (default: `Middle Mouse`)
- Smooth **ease-in / ease-out** animation (`200ms`)
- Camera remains **fully rotatable** while zoomed (no camera lock)
- Works **on foot** and **in vehicles**
- First-person mode uses **head offset correction** to prevent clipping
- Zero performance impact (very light native calls)

---

## Controls
| Action      | Default Key         | Behavior |
|------------|---------------------|---------|
| Hold Zoom  | `MOUSE_MIDDLE`      | Zooms in while held, returns when released |
